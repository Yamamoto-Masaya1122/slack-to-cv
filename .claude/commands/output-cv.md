---
description: storage/配下の最新日時フォルダから職務経歴書を生成する
---

`storage/` 配下にあるフォルダ名が最大（=最新日時）のフォルダを1つ特定し、その中の `{channel_name}_messages.json` と `resume_draft.md` を分析して職務経歴書を生成してください。

## 手順

1. 最新フォルダを特定する（フォルダ名は日時文字列なので、名前で降順ソートした先頭が最新）:
   ```bash
   ls -d storage/*/ | sort | tail -1
   ```
2. そのフォルダ内の `resume_draft.md` と `*_messages.json` を読み込む。
   - どちらかが存在しない場合は、その旨を報告して中断する。
3. `resume_draft.md` に記載された「生成ルール」「制約」「プロジェクト情報」を厳守する。
   - 特に: STAR構成の地の文・全体400文字以内・Slackメッセージから読み取れない情報は捏造しない。
4. `*_messages.json` の `messages` 配列の内容を根拠に、`▼担当業務` と `▼詳細` を生成する。
5. 生成結果を**同じフォルダ内に `resume.md` という新規ファイル**として出力する。`resume.md` の中身は以下の構成とする:
   - `## プロジェクト情報`（`resume_draft.md` のプロジェクト情報をそのまま転記）
   - `## プロジェクト詳細・担当業務・主な実績`（生成した `▼担当業務` と `▼詳細`）
6. 生成した内容をMarkdown形式でターミナルにも表示する。

## 注意
- メッセージから定量的な成果（数値・割合・時間）が読み取れる場合は積極的に活用する。
- 推測が必要な場合は合理的な範囲にとどめ、断定的な捏造はしない。
